## Usage

As a prerequisite it's assumed you have npm/yarn installed and the gulp-cli.

* Clone the repo: 

```
https://github.com/sreenathkspanikker/Gulp4-Bolilerplate.git
```

* Install dependencies

```
npm install / yarn
```

* Start gulp

```
gulp
```
